## 1. Claude とは？

Claude は、Anthropic が開発した生成 AI チャットサービスです。自然な日本語対応、高い安全性、そして実務で役立つ多彩な機能が特徴です。

| 項目 | 詳細 |
|------|------|
| 開発元 | Anthropic |
| 最新モデル | Claude Opus 4.6 / Sonnet 4.6 / Haiku 4.5 |
| 対応言語 | 日本語を含む多言語 |
| アクセス方法 | Web・モバイルアプリ・API・Claude Code |

## 3. 主要機能 20 選

### 基本チャット機能

#### 機能 1｜高精度な日本語チャット
自然な日本語で質問・指示ができます。文脈を理解した長い会話も得意で、ビジネス文書の作成から日常的な相談まで幅広く対応します。

**使い方の例：**
```
「以下のメールに対する返信文を、丁寧なビジネス語調で作成してください。」
```

---

#### 機能 2｜長文コンテキスト処理（最大 20 万トークン）
一度に約 15 万字（日本語）もの長文を処理できます。長い契約書・調査レポート・会議資料を丸ごと読み込ませて分析・要約が可能です。

**活用例：**
- 数十ページの PDF を要約
- 複数の資料を比較分析
- 長い議事録から要点を抽出

---

#### 機能 3｜ウェブ検索（Search）
リアルタイムでウェブを検索し、最新情報をもとに回答を生成します。知識カットオフを超えた情報も取得可能です。

**使い方：** 設定からウェブ検索をオンにするだけで自動的に活用されます。

---

#### 機能 4｜拡張思考（Extended Thinking）
複雑な問題に対して、Claude が内部で段階的に推論するモードです。数学・論理問題・多段階分析などで特に効果を発揮します。

---

### コンテンツ生成・編集

#### 機能 5｜文書作成・編集
メール・報告書・プレゼン原稿・SNS 投稿など、あらゆる文書を生成・リライトできます。

**対応フォーマット：**
- ビジネスメール / 社内文書
- ブログ・SNS 投稿
- プレゼン資料の構成案
- 学術文書・レポート

---

#### 機能 6｜コード生成・デバッグ
Python・JavaScript・SQL など多様なプログラミング言語のコードを生成します。既存コードのエラー発見・修正提案も得意です。

**使い方の例：**
```
「Excelから読み込んだCSVを整形して、グラフを作るPythonスクリプトを書いてください。」
```

---

#### 機能 7｜スタイル機能（文体設定）
自分の文章スタイルを学習させ、Claude がその文体で文章を生成するよう設定できます。SNS 運用や継続的な文書作成に便利です。

**設定方法：** 設定 → スタイル → サンプル文章を貼り付けて学習

---

#### 機能 8｜アーティファクト（Artifacts）
コード・HTML・React コンポーネント・SVG・Markdown などを、その場でレンダリング・プレビューしながら作成できる機能です。

**作れるもの：**
- インタラクティブな Web アプリ
- データ可視化グラフ
- SVG イラスト
- Markdown ドキュメント

---

### ファイル・データ処理

#### 機能 9｜ファイルアップロード・分析
PDF・Word・Excel・画像などのファイルをアップロードして分析できます。

| ファイル種別 | できること |
|------------|------------|
| PDF / Word | 要約・翻訳・校正 |
| Excel / CSV | データ分析・グラフ生成 |
| 画像 | 内容説明・OCR・分析 |
| コード | レビュー・リファクタリング |

---

#### 機能 10｜ファイル生成（Word / Excel / PowerPoint）
チャット上で Word 文書・Excel スプレッドシート・PowerPoint スライドを生成して、そのままダウンロードできます。

---

#### 機能 11｜Deep Research（ディープリサーチ）
特定のテーマについて複数のウェブソースを横断的に調査し、詳細なリサーチレポートを自動生成します。

**使い方：** チャット画面の「リサーチ」ボタンをクリックしてテーマを入力

---

### プロジェクト管理

#### 機能 12｜プロジェクト機能（Projects）
テーマ別に会話・ファイル・カスタム指示をまとめて管理できます。

**設定できること：**
- プロジェクト専用のカスタム指示（システムプロンプト）
- 共通ファイル・資料の登録
- チームでの共有（Team プラン以上）

**活用例：**
```
「法務レビュー用Claude」→ 自社契約テンプレート＋法務ガイドラインを登録
「マーケ用Claude」→ ブランドガイドライン＋ペルソナ情報を登録
```

---

#### 機能 13｜メモリ機能（Memory）
過去の会話から重要な情報を記憶し、次の会話に自動的に反映します。毎回説明し直す手間がなくなります。

**記憶される情報の例：**
- 名前・所属・職業
- よく使うスタイルや好み
- プロジェクトの背景情報

---

#### 機能 14｜Skills（スキル）
よく使う作業手順をパッケージ化して登録し、ワンコマンドで呼び出せます。繰り返し作業の効率化に最適です。

**例：**
```
「/会議録作成」→ 定型フォーマットで議事録を自動生成
「/週次レポート」→ 決まった構成でレポートを作成
```

---

### 外部サービス連携

#### 機能 15｜コネクタ（MCP 連携）
Gmail・Google カレンダー・Google Drive・Slack など外部サービスと連携し、Claude から直接操作できます。

**主な連携サービス：**
- **Google Workspace**: Gmail・Drive・Calendar
- **Microsoft 365**: Excel・Word・PowerPoint・Outlook（ベータ）
- **コミュニケーション**: Slack など
- **その他**: 300 以上の MCP サービス

**例：**
```
「今週の予定を確認して、明日の会議の議題をGoogleカレンダーに追加して」
「Gmailの未読メールを要約して、重要なものだけ教えて」
```

---

#### 機能 16｜Microsoft 365 アドイン（Excel / PowerPoint / Word）
Excel・PowerPoint・Word のアドインとして Claude を呼び出し、各アプリ内で直接 AI 支援を受けられます。（2026 年 5 月 GA）

---

### 高度な自動化機能

#### 機能 17｜Claude Code
ターミナル（コマンドライン）から Claude を操作し、コーディング・ファイル操作・スクリプト実行などを自律的に行うエージェント型ツールです。

**主な用途：**
- コードの自動生成・修正・テスト
- ファイルの一括処理
- 複雑なワークフローの自動化
- MCP を通じた外部サービス操作

**インストール：**
```bash
npm install -g @anthropic-ai/claude-code
```

---

#### 機能 18｜Claude Cowork（コワーク）
Claude がパソコンの Chrome ブラウザを直接操作するデスクトップ機能です。Web 上の操作を自動化できます。（2026 年 4 月 GA）

**使い方：** Claude デスクトップアプリ → 上部「Cowork」ボタン

**活用例：**
- YouTube Studio でのタイトル・説明文・タグの一括設定
- Web フォームへの自動入力
- オンラインシステムの定型操作

---

#### 機能 19｜Claude in Chrome（ブラウザエージェント）
Chrome 拡張として動作し、閲覧中のページを Claude が認識・操作するブラウジングエージェントです。

---

#### 機能 20｜ゲストパス
Max プラン以上のユーザーが、友人・同僚に 7 日間の Pro 相当アクセスを無料でプレゼントできる機能です。

**使い方：** 設定 → ゲストパス → リンクを発行して共有

Claude は単なるチャット AI にとどまらず、**ファイル処理・外部サービス連携・ブラウザ操作・コード自動化**まで幅広く対応するプラットフォームへと進化しています。
